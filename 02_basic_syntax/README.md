## Quick Reference

```python

name = "Alice"

# Numbers
integer = 10
floating = 3.14
complex_num = 3 + 4j

# Strings
text = "Hello"
formatted = f"Value: {x}"

# Type hints
def greet(name: str) -> str:
    return f"Hello, {name}"
```
