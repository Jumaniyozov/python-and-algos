# Chapter 1: Getting Started with Python

## Learning Objectives

By the end of this chapter, you will:
- Install Python 3.14+ on your system
- Understand the Python execution model
- Use the Python REPL (interactive shell)
- Write and run your first Python programs
- Know what's new in Python 3.14
- Set up a productive development environment

   - REPL sessions and commands

3. **tips.md** - Best practices and gotchas
   - Common installation issues
   - REPL productivity tips
   - Choosing the right execution mode

## Quick Start

```bash
# Verify Python installation
python --version  # Should show 3.14.x or higher

# Start the REPL
python

# Run a Python script
python hello.py
```
