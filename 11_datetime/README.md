# Chapter 11: Date and Time

## Overview
Master Python's datetime handling capabilities, including timezone management, date arithmetic, formatting, and time-based calculations.

## Learning Objectives
- Work with dates, times, and timedeltas
- Handle timezones correctly
- Parse and format datetime strings
- Perform date arithmetic
- Use calendar and time modules

## Topics Covered
1. **datetime Module** - date, time, datetime, timedelta
2. **time and calendar** - Timestamps, calendar operations
3. **Timezones** - zoneinfo, UTC, timezone-aware datetimes
4. **Formatting and Parsing** - strftime, strptime
5. **Common Patterns** - Date ranges, business days, age calculation
