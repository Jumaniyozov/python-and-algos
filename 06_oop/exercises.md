# OOP: Exercises

## Exercise 1: Rectangle Class
Create a `Rectangle` class with width/height. Add methods for area and perimeter.

## Exercise 2: Inheritance
Create `Vehicle`, `Car`, and `Motorcycle` classes with appropriate attributes and methods.

## Exercise 3: Magic Methods
Implement a `Fraction` class with `__add__`, `__sub__`, `__mul__`, `__truediv__`.

## Exercise 4: Properties
Create `Circle` class with radius property and computed area/circumference properties.

## Exercise 5: Dataclass
Convert a traditional class to use `@dataclass`.

## Exercise 6: Abstract Base Class
Create `PaymentMethod` ABC with `process_payment` method. Implement `CreditCard` and `PayPal`.

## Exercise 7: Context Manager
Implement a class-based context manager for file operations.

## Exercise 8: Descriptor
Create a `ValidatedString` descriptor that validates string length.

## Exercise 9: Protocol
Define a `Serializable` protocol with `to_dict` and `from_dict` methods.

## Exercise 10: Composition
Build a `Library` class that contains `Book` objects.
